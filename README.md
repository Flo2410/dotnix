# dotnix

This repository contains my NixOS config.

## Profiles

### x86_64

- **fwf**: My Framework 13 (AMD Ryzen AI 7 350) daily-driver.
- **pc-florian**: My Desktop PC (AMD Ryzen 9 9950X3D and NVIDIA GeForce RTX 3080).
- **surface**: My Surface Pro 4 used as a tablet for controlling Home Assistant. (unmaintained)
- **helios**: My VPS config. (WIP)

### aarch64

> Some info on setting up NixOS on a Raspberry Pi Zero 2 W can be bound at <https://github.com/plmercereau/nixos-pi-zero-2?tab=readme-ov-file>

- **curiosity**: My carry-on Raspberry Pi 4. (unmaintained)
- **ups-pi**: Runs a [NUT](https://networkupstools.org/) server to monitor a UPS.

### Other

- **wsl**: Config to use NixOS in WSL. (unmaintained)
